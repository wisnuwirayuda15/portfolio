# Portfolio Website Showcase

![Portfolio Website Cover Image](https://raw.githubusercontent.com/wisnuwirayuda15/portfolio/main/public/img/portfolio-web.png "Portfolio Website Cover Image")

Halo teman-teman semuanya, dan tentunya juga sepuh kita Pak Sandhika Galih :D. Pada kali ini saya mau share personal project pertama saya di sesi showcase project ini, yaitu Website Porfolio. Project ini yang saya bikin kurang lebih selama 2 mingguan. Projek ini sebenernya awalnya iseng-iseng buat belajar TailwindCSS, tapi kok lama-lama ketagihan ya wkwkwkw, jauh lebih enak dari Bootstrap yang biasa saya pake.
